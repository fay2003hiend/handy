Detailed Description

template<bool StartOnCreation = true>
class handy::impl::Benchmark< StartOnCreation >

A class to do precise benchmarking on both Windows and Linux.

Template Parameters
StartOnCreationbool template constant telling if the clock initialization must be done at creation

If StartOnCreation is true, this class initializes the clock on creation. You can also reset it by calling the start() function. The finish() function returns the time (in seconds, up to nanoseconds of precision) elapsed since the call to the start() function.
