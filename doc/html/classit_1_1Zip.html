Detailed Description

template<typename... Containers>
class it::Zip< Containers >

This class servers mainly as a wrapper for iterating in the for range loop. It is composed of a tuple of references to containers that are iterable, and exposes a 'begin' and 'end' methods, returning a 'ZipIter' with the iterators of each container passed as argument.
