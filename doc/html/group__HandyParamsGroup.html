Simply call this guy at the public section of your class, passing the class name as first argument and a sequence of the arguments you want to be capable of initialization. For examples of usage, see the 'examples' folder.

Warning
The types passed by parameter must be exactly equal to the ones in the class. That is, if theres a float in your class, and you pass a double, std::any will throw an exception.
