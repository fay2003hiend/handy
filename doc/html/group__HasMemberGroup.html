std::integral_constant

Uses SFINAE to determine the existence of a certain member given a class.

Given a call to HAS_VAR(), HAS_FUNC(), HAS_EXTERN_FUNC() or HAS_OVERLOADED_FUNC(), this macro will generate two functions and one class. By using SFINAE, we can determine if a given class has or has not a member variable or function at compile time.

Note
I am using HM_T and HM_Args to avoid any naming conflict if you call this guy inside a class template, for example.
