The sole reason these functions were defined is to allow pointers to be called in 'zip' as if they were iterable types, having a starting and ending points. As the first argument defines the range, there is no problem in passing a pointer to be iterated, as long as its range is smaller than the range of the first argument of the 'zip' function.
