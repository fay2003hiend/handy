Detailed Description

template<class BaseType>
struct cnt::help::Accessor< BaseType >

We inherit from either 'Container' or 'Slice'.

The only purpose of this class is to delegate calls to the accessors of 'Container' or 'Slice', so we dont have duplication of code and the classes can be written more clearly.
