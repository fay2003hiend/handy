Detailed Description

template<typename T, std::size_t N = 0>
struct cnt::Vector< T, N >

'cnt::Vector' inherits from 'std::vector<T>' if it is not supplied with compile time size or if the given compile time size is greater than the value defined at 'cnt::help::maxSize', for maximum stack size allocation. Otherwise it inherits from 'std::array<T, N>'.
