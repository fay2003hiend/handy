When using stl functions which takes iterator parameters of the form (first, last), these macros make it much easier. Simply use ZIP_ALL(container) for a container class that is iterable (that is, has std::begin and std::end definitions or is a pointer). These macros simply call it::help::begin and it::help::end for each argument, delegating the call to std::begin and std::end while also supporting pointer types to be called.
