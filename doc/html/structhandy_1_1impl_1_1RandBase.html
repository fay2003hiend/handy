Detailed Description

The base class for generating random numbers.

The class has only a generator variable (generator) and a function for setting the seed value (seed) If you pass a -1 (basically the same thing as std::string::max_size()), it will call std::random_device{}(), which will create the seed (pseudo-) randomly.
