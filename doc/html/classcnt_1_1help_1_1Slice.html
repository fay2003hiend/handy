Detailed Description

template<class Cnt>
class cnt::help::Slice< Cnt >

This class is just a proxy to access chosen dimensions of a container. The only storage is are the positions to access and a reference to the container that created this slice. There are a bunch of examples of how to use it in 'examples/SliceExamples.cpp'.
